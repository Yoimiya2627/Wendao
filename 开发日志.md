**chapter1.json**

### story_phase 完整状态机
- phase 0：游戏启动，在ShopScene，morning对话未触发
- phase 1：morning结束，玩家可出门探索
- phase 2：玩家进入测灵广场，test对话触发中
- phase 3：测试结束，回程模式激活，强制触发节点生效
- phase 4：废庙战斗触发
- phase 5：战斗结束，after_battle对话结束
