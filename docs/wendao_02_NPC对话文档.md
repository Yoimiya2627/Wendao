## 苏明（爹）

### 早晨（morning场景已有，保持不变）
关键台词："不管测出什么，记得回来吃饭。"

### 破碗互动（新增，杂货铺室内可选交互）
```

旁白：一只碗，碗边有一道细细的裂缝，补了一半，线头还留在那里。
旁白：她伸手想碰一碰。
苏明：别动那个。
旁白：爹头也没抬，语气很平。

旁白：苏云晚把手收回来，没有问是谁。

```

### 回家盛饭（新增，强制触发）
```
旁白：爹在灶台边，听见门响，没有回头。
旁白：锅里的汤还热着，咕嘟咕嘟地响。
苏明：回来了。
旁白：他盛了碗饭，放在桌上，又盛了碗汤，放在饭旁边。
旁白：没有问今天怎么样。
苏云晚坐下来，低着头。
苏明：吃吧，热的。
旁白：她拿起筷子，没有说话。
旁白：灯芯烧得很小，火苗一点一点地跳。屋子里只有锅里汤的声音。
旁白：她没有哭。就是坐在那里，吃完了那碗饭。
旁白：灶台边挂着一件旧围裙，不是爹的，也不是她的，洗得发白，已经很久没人动过了。
```

## 妖兽战斗设计说明

妖兽没有任何情绪，不是恨云晚，只是在清理障碍。
战斗文本体现冷漠感，不是恶意的化身，而是规则的清道夫。
这和天道"傲慢的漠视"的世界观完全吻合。
