# 《问道》游戏项目完整设计文档

## 基本信息

- **游戏名称**：问道
- **引擎**：Godot 4
- **平台**：华为手机（Android）+ 电脑
- **画面**：横屏，1280×720视口
- **操控**：虚拟摇杆（手机左下角）+ 方向键/WASD + E键交互（电脑）
- **制作人**：伟鑫
- **受众**：王薇（女朋友，INFJ，喜欢剑来和仙逆）
- **制作原因**：一周年纪念礼物，专门为她量身定制

---

## 世界观

**世界名称**：九洲仙墟

**核心设定**：修仙界靠灵根决定命运，天道设下规则，灵根品阶高低决定一个人能走多远。普通人无灵根就等于没有修仙资格，被社会边缘化。天道如行人，凡人如蝼蚁——天道不是邪恶的，只是傲慢到看不见凡人。

**灵感来源**：仙逆的逆天灵根体系 + 剑来的小人物叙事节奏和人情世故厚重感。原创世界观，致敬两个IP的气质。

## 主要人物设定

### 苏云晚（女主）
- 16岁，凡人小镇杂货铺女儿
- 灵根测试显示"无灵根"，实际拥有万年未现的"混沌灵根"——可兼修所有属性法则，是天道的一个BUG，连天道自己都没料到
- 外表冷静，嘴硬心软，善良但很犟
- 她的"犟"是向内的——守住自己的东西，不让外面的世界侵犯进来

### 苏明（爹）
- 表面：话很少，种菜养鸡，普通凡人父亲
- 实际：隐藏大佬，参照剑来齐静春的感觉——平平无奇，实则深不可测
- 第一章完全是普通父亲形象，后续章节再一点点揭开
- 深爱女儿，不善表达，用行动代替语言
- 关键台词："不管测出什么，记得回来吃饭。"
- 关键台词："别动那个，等她回来自己补。"（关于破碗）

### 苏云晚的妈妈（暂未命名）
- 第一章只有两个伏笔：
  1. 灶台边挂着一件旧围裙，洗得发白，很久没人动过
  2. 一只补了一半的破碗，线头还留在那里
- 她不是普通人，和爹当年在同一个世界
- 她的"犟"是向外的——要去改变外面的世界，找天道讲理
- 她发现了一件不公平的事，觉得不对劲，锅里炖着鱼，围裙随手一搭，以为去讲个理马上回来
- 她低估了规则的傲慢，一耗就是十六年
- 她找到云晚时能说的只有："我以为我很快就回来的。"
- 她的执念和性格传给了云晚——母女两个都是同一种犟

### 年年（布偶猫）
- 性格：极度温顺，安静，来者不拒，跟随王薇最久，世界上最好的小猫
- 不乱跑乱跳，你怎么撸都来者不拒
- 游戏里：安静趴着，把头靠进云晚手心，看着她出门；回家后安静坐在脚边陪着她

### 大鱼（缅因猫）
- 性格：活泼跳脱，比较大只，会欺负年年，跟王薇也会打架
- 游戏里：从后院蹦出来，叼来旧剑穗，大摇大摆走开；回家后用巨大脑袋用力蹭腿

### 顾飞白
- 落魄剑修，废庙战斗后出现
- 他的出现不是巧合，他知道太平宗在做什么，早就知道那夜废庙会发生什么
- 台词："有人动了手脚。也意味着，你走的路，或许比那些五灵根的人更难——也更远。"

### 太平宗
- 地方宗门，第一章例行公事来碎玉镇测灵根
- 完全没发现苏云晚的异常（混沌灵根超出他们的认知范围，就像只会算术的人看到微积分）
- 测验师宣布"无灵根"是真的不知道，不是隐瞒
- 第二章开始察觉废庙异动，介入调查，后续成为重要势力

## 地图设计

### 基本参数
```
地图尺寸：40×30格
每格大小：32×32像素
地图总像素：1280×960px
视口大小：1280×720px（横屏）
```

### 布局
环形布局，中央老榕树为核心：

- **左上象限**：苏家杂货铺（出生点，独立室内场景）
- **右上象限**：测灵广场（大地图无缝，青石板铺地，道纹石碑）
- **左下象限**：碎玉集市（大地图无缝，摊位错落）
- **右下象限**：悠然茶馆+废庙（茶馆独立室内场景，废庙独立场景）
- **中央**：老榕树+古井+公告栏+老狗
- **南门**：守卫把守

### 场景结构
- 大地图无缝：中央广场、测灵广场、碎玉集市
- 独立切屏：杂货铺室内、茶馆室内、废庙
- 进入建筑：走到门口按E键，屏幕显示"按E进入"提示

## 第一章剧情流程（约31分钟）

### 一、杂货铺早晨（约2分钟）
- morning对话自动触发
- 年年安静趴着，把头靠进云晚手心，看着她出门
- 大鱼从后院蹦出来，叼来旧剑穗放在脚边，大摇大摆走开
- 云晚随手把剑穗揣进口袋
- 爹说："不管测出来什么，记得回来吃饭。"
- 可选互动：破碗（爹说"别动那个，等她回来自己补"，交互图标消失）
- 走到门口按E出门

### 二、出门逛集市（约4分钟）
- 路过公告栏（可选）
- 路过古井，和打水人对话（可选）
- 路过老狗（可选）
- 路过两个大婶，她们热络打招呼
- 走到集市，遇算命先生（可选，埋伏笔）
- 集市深处遇老婆婆，触发market对话（强制）
- 可选：走到茶馆门口和掌柜说话

### 三、测灵广场（约4分钟）
- 走到广场触发test对话
- 灵根测试失败，玩家选择：没什么大不了的 / 我不信
- 测验师正常宣布结果，没有任何异常反应

### 四、测试失败走回家（约7分钟）——路C核心段落
回程路线和去程不同，依次强制触发：
1. 庆祝的少年——路过集市边缘，同龄少年被家人围着庆祝，两人对视一秒，少年尴尬移开视线
2. 两个大婶——声音压低了但没压够，苏云晚听见了，她们也知道她听见了，大家都没说破
3. 打水人——看见她，想说什么，最后低下头专心看井里的水
4. 算命先生——收摊，平静看她一眼，继续收
5. 老狗——听见脚步声，抬了抬眼皮，继续睡

回到杂货铺室内强制触发：
- 爹背对着她在灶台边，听见门响没有回头
- 默默盛了碗饭放在桌上，说："吃吧，热的。"
- 年年走到她脚边，安静坐着陪着她，把头轻轻靠进她掌心
- 大鱼用巨大脑袋用力蹭她的腿，差点把她顶歪
- 灶台边挂着一件旧围裙，洗得发白，很久没人动过（妈妈伏笔）
- 苏云晚没有哭，把那碗饭吃完了

### 五、夜晚降临（约2分钟）
- 屏幕缓慢变暗，旁白交代天色已晚
- 废庙方向出现异动，引导玩家走过去
- 走到废庙门口按E进入

### 六、废庙探索+激活混沌灵根（约7分钟）
- 废庙里有四块散落的道纹碑文，玩家主动走过去按E阅读
- 读完全部碑文后，完整石碑发光
- 玩家主动走过去按E触碰，激活混沌灵根
- 战斗触发

### 七、战斗+章末选择（约5分钟）
- 战斗结束，触发after_battle对话
- 遇到顾飞白
- 章末选择：一起走 / 先回去看爹
- 选先回去：发现爹留了信（letter场景）

## 叙事风格

- 自由探索为主，INFJ友好
- 飘字不锁定移动
- 情感高潮节点锁定1-2秒，配合音乐
- 对话逐字显示，0.04-0.05秒/字，点击加速
- 苏云晚的台词要有她自己的性格：冷静、嘴硬、善良但很犟
- 留白优于解释，玩家感受到就够了

## 技术防Bug规范

### NPC交互
- 射线检测，按E触发
- 同一时间只允许一个对话激活
- 对话冷却0.5秒
- 距离检测：玩家和NPC距离≤40px才显示交互提示

### 场景切换
- 每个场景必须有SpawnPoint节点标记出生点
- 切换后玩家在SpawnPoint位置，禁止默认(0,0)
- 场景切换冷却1秒，防止重复触发
- 所有切换使用SceneTransition，有淡入淡出效果

### story_phase管理
- 只能+1递增，统一用advance_phase()函数
- 禁止直接赋值
- 每个剧情节点有is_triggered布尔值，触发后不重复

### 碰撞层
- Layer0：地面（可走）
- Layer1：碰撞墙（不可走）
- 严格分离，禁止混用
- 碰撞体厚度≥8px

### 摄像机
- Camera2D设置Limit四边界
- 严禁越出地图范围
- 不能出现黑边

### 对话框
- 单行最多20个中文字符，自动换行
- 对话时虚拟摇杆自动隐藏
- 对话结束0.5秒冷却

### 分辨率适配
- 所有UI使用锚点布局
- Godot项目设置开启拉伸模式
- 支持华为手机分辨率

### 存档
- 先写临时文件再替换正式存档
- 存档包含所有NPC的is_triggered状态
